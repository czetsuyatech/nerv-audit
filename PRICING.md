# nerv-audit Licensing & Pricing (v1.0)

## Overview

nerv-audit is a developer-focused auditing library built on top of Hibernate Envers, providing
enhanced entity versioning, field-level diffing, and advanced audit querying capabilities.

This document outlines the official licensing and pricing model for nerv-audit.

---

## Editions

### 1. nerv-audit-lite (Free)

* Publicly available
* Includes:

    * Basic entity versioning (horizontal/vertical auditing)
    * Envers integration and simplified configuration
    * Basic revision history access
      * horizontal 
      * vertical (limited to the first five audits)

This edition is intended for evaluation, small projects, and general use.

---

### 2. nerv-audit-core (Commercial)

* Private distribution
* Includes:

    * Field-level change tracking (horizontal/vertical auditing)
    * Advanced audit querying API
    * Performance optimizations
    * Enhanced developer experience
